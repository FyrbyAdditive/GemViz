# GemViz

A macOS app for previewing GemCAD `.gem` design files with 3D visualization.

## Features

- 3D preview of GemCAD gem designs
- Quick Look extension for Finder previews (press Space on any `.gem` file)
- Configurable surface opacity, reflectivity, and color
- Settings in the app apply to Quick Look previews

## Installation

Install `GemViz.app` to `/Applications` using the package installer from Releases and launch it once to register the Quick Look extension.
